# Sustania - Digital Farming Solutions 🌱

![Sustania Logo](src/img/logo.png)

A smart irrigation system and comprehensive digital farming solution designed to help farmers optimize water usage, improve crop yields, and make data-driven decisions.

## 🎯 Project Overview

Sustania is a digital farming platform that combines IoT sensors, machine learning, and a user-friendly dashboard to revolutionize agricultural practices. The system provides real-time monitoring of farm conditions, automated irrigation control, and data-driven insights for optimal farming decisions.

### Key Features

- **Smart Irrigation System** 💧
  - Real-time moisture monitoring
  - Automated zone control
  - Customizable scheduling
  - Water usage analytics

- **Weather Integration** ⛅
  - 5-day weather forecasts
  - Temperature monitoring
  - Wind speed tracking
  - Humidity data

- **Community Forum** 👥
  - Knowledge sharing
  - Market price discussions
  - Technical support
  - Best practices exchange

- **Analytics Dashboard** 📊
  - Water usage trends
  - Crop performance metrics
  - Cost analysis
  - Optimization recommendations

## 🚀 Quick Start

1. Clone the repository
```bash
git clone https://github.com/RonnMath03/Sustania.git
```

2. Install dependencies
```bash
cd sustania
npm install
```

3. Set up environment variables
```bash
cp .env.example .env
# Edit .env with your configuration
```

4. Start the development server
```bash
npm run dev
```
